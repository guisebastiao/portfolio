# 🚀 Portfolio

Um site de portfólio moderno e de alto desempenho, desenvolvido com as tecnologias web mais recentes para exibir projetos e experiência profissional.

## 🌐 Live Demo

Acesse a versão em produção:

**https://guisebastiao.com.br**

## 🛠️ Tech Stack

**Core:**

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilo:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações:** [Motion](https://motion.dev/) & [Lenis](https://github.com/darkroomengineering/lenis) (Smooth Scroll)

**Libraries & Tools:**

- **I18n:** `i18next` & `react-i18next` (Suporte multilíngue)
- **Forms:** `react-hook-form` & `zod` (Validação)
- **UI Components:** Base UI, Shadcn UI, & Lucide React
- **Communications:** [EmailJS](https://www.emailjs.com/)
- **Theme:** `next-themes` (Dark/Light mode)
- **Toasts:** `sonner`

## ✨ Features

- 🌍 **Internationalization:** Suporte completo para os idiomas português brasileiro e inglês americado.
- 🌓 **Theme Switching:** Transição fluida entre os modos claro e escuro.
- 📱 **Responsive Design:** Layout totalmente adaptável para dispositivos móveis, tablets e desktops.
- 📈 **Analytics:** Rastreamento de eventos integrado para interações de projetos.
- ✉️ **Contact Form:** Seção de contato totalmente funcional, utilizando o EmailJS.
- 🎨 **Modern UI:** Animações fluidas e rolagem suave para uma experiência do usuário aprimorada.

## 📂 Estrutura do Projeto

```text
src/
├── app/            # Next.js App Router (Pages, Layouts, Providers)
├── assets/         # Assets estáticos (Imagens, Dados, PDFs)
├── features/       # Feature-based modules (About, Contact, Projects, etc.)
│   └── [feature]/
│       └── components/
└── shared/         # Components compartilhados, hooks, utilities, e providers
```
